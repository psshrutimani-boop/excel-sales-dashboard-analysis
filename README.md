# Excel Sales Data Analysis Project

## Project Overview
This project analyzes sales data using Microsoft Excel.

Tools Used:
- Microsoft Excel
- Pivot Tables
- Charts

## Analysis Performed
1. Sales by Region
2. Sales by Category
3. Sales by Segment
4. Top 10 Products by Sales
5. Profit by Region

## Key Insights
- Identified top performing regions
- Analyzed customer segments
- Found highest selling products
- Compared profit across regions
